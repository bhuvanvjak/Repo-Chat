GitReader Setup

GitReader uses Groq for fast, free LLM chat — you'll need your own Groq API key (it's free, takes about 30 seconds, no credit card). The key is stored only in your browser and sent only to your own local GitReader backend.

1. Get a free key

Open console.groq.com/keys, sign in, and click "Create API Key".

2. Paste it here

You can change this anytime — right-click the GitReader icon → Options, or use the Settings button in the side panel.
